Вариант 1 (идеальный: у вас macOS и Xcode):
— Зайти в терминале в папку с файлом Makefile.for_xcode и выполнить команду
  make -f Makefile.for_xcode
— Можно открывать файл Mashgraph_Task3_OpenGL.xcodeproj

Вариант 2 (тоже норм: у вас macOS, Xcode не установлен, но зато есть Command Line Tools):
— Зайти в терминале в папку с файлом Makefile.macOS и выполнить команду
  make -f Makefile.macOS

Вариант 3 (надеюсь, сработает… у вас Unix-подобная система и установлены библиотеки GL, glut, GLEW):
— Зайти в терминале в папку с файлом Makefile и выполнить команду
  make
